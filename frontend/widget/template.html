Embed Instructions

Add this single <script> tag to your page:

<script
  src="https://your-domain.com/widget/embed.js"
  data-partner="demo-partner"
  data-brand-color="#1a1a2e"
  data-logo="https://your-logo.png"
  data-position="right"
  data-greeting="Ask me anything about cricket rules!"
></script>

Configuration Options

AttributeDefaultDescription
data-partnerunknownPartner ID for API auth
data-brand-color#1a1a2ePrimary brand color
data-logo—Partner logo URL (header)
data-positionrightright or left
data-greetingAsk me...Initial bot greeting
data-placeholderAsk a cricket...Input placeholder
